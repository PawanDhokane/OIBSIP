## 🍕 Level 3 – Pizza Delivery Application

A full-stack Pizza Delivery Web Application.

### Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB

### Features
- User Authentication (Login / Register)
- Browse and Order Pizzas
- Cart Management
- Admin Dashboard
- Order Management

---

## 🚀 How to Run (Level 3 Project)

### Client
cd client
npm install
npm run dev

### Server
cd server
npm install
npm start
